# 🧠 Spelling & Grammar Checker

A modern, responsive web-based spelling and grammar checker built using **HTML, CSS, and JavaScript**. This Website allows users to quickly detect and correct writing mistakes with an intuitive and visually appealing interface.

---

## ✨ Features

- 🔍 **Real-time Grammar & Spell Checking** using LanguageTool API  
- 🎯 **Clickable Suggestions** to instantly fix errors  
- 🎨 **Modern Glassmorphism UI** with smooth animations  
- ⚡ **Fast & Lightweight** – no frameworks required  
- 📱 **Responsive Design** for desktop and mobile use  

---

## 🚀 How It Works

1. Enter or paste your text into the input box  
2. Click the **"Check Now"** button  
3. View highlighted errors along with suggested corrections  
4. Click on a suggestion to automatically replace the incorrect word  

---

## 🛠️ Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling & animations  
- **JavaScript (Vanilla)** – Logic & API handling  
- **LanguageTool API** – Grammar and spell checking
